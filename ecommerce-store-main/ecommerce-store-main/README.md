# Rwandan Traditional Attire Store

Fully responsive fictitious ecommerce website for "Rwanda Trad" - a boutique store based in Rwanda specializing in traditional attire such as Mushanana, Umutara, and Agaseke crafts.  

Fully functional frontend, would have to integrate with a backend at some point to make it an actual working site where people can place fake orders and what not.  
All prices are displayed in Rwandan Franc (RWF).

Users can neither Login nor Register in the account section atm.
